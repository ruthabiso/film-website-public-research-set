# film-website-public-research-set
该仓库主要用于建立影视类网站的公开研究资料集合，对互联网上可访问的电影与电视剧站点进行基础信息整理。整理内容包括站点名称、访问链接、站点类型说明及简要定位描述，方便统一管理与长期维护。本项目不提供影视内容服务，也不涉及资源传播、播放或技术解析，仅作为站点公开信息的研究性整理集合存在。所有资料均整理自公开网络环境，与所记录站点不存在任何隶属、合作或商业关联关系。
